# NeteaseCloudMusic

#### 基于React和mobx的网易云音乐App

> 后端使用网易云音乐API的
>
> github地址：https://github.com/Binaryify/NeteaseCloudMusicApi

#### 技术点

> React、React-router、mobx、fetch、ant-mbnile、less、ES6

#### 功能的实现

- 歌单页面、推荐、歌手、排行榜
- 搜索歌手，歌名等
- 音乐播放器
- 视频播放、歌词展现

#### 项目的启动

+ 网易云接口

  ```shell
  # 在 NeteaseCloudMusicApi 目录中执行
  	cnpm install
  	node app.js   # 启动
  ```

+ React

  ```shell
  #在 react-music-msater 目录下执行
  	cnpm install -g yarn	
  	yarn install
  	yarn start	
  ```
